Fortinet SSL VPN Clipboard

Skip the repetitive Fortinet clipboard clicks.

These bookmarklets automate the clipboard panel inside the Fortinet SSL-VPN web UI. Save them once, then use a single click instead of stepping through the same clipboard flow every time.

The manual process is usually: press F8, open Clipboard, request or paste text, press the copy button, press Clear, and close the slideover. These bookmarklets take care of the repeated clipboard actions for you while that Fortinet page is open.

  1. 1. Install themDrag each button below to your bookmarks bar once.
  2. 2. Open FortinetKeep your current Fortinet SSL-VPN page open in the browser.
  3. 3. Click oncePress ↑ Send clipboard or ↓ Get VPN clipboard whenever you need it.

↓ Get VPN clipboard

Fetch the shared Fortinet clipboard text and copy it into your local clipboard.

Read text from your local clipboard and push it to the shared Fortinet clipboard field.

If the browser asks for clipboard permission, allow it for the Fortinet page. The ↑ Send clipboard bookmarklet needs clipboard read access.

↑ Send clipboard moves text from your computer into the remote session. ↓ Get VPN clipboard brings the remote clipboard back to your local machine.
